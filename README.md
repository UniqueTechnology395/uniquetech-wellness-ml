# Wellness Prediction AI via Decision Trees

Αυτό το project αποτελεί μέρος του βραβευμένου βιο-ρομποτικού συστήματος της Unique Tech (ACSTAC 2026).

Πώς λειτουργεί:
Training: Εκπαιδεύουμε ένα μοντέλο DecisionTreeClassifier στην Python χρησιμοποιώντας δεδομένα CO2, PM2.5 και Heart Rate.

Export: Χρησιμοποιούμε την export_text για να εξάγουμε τη λογική του δέντρου σε ανθρώπινη γλώσσα (Explainable AI).

Deployment: Μεταφέρουμε τους κανόνες στο Arduino Uno Q για λήψη αποφάσεων σε πραγματικό χρόνο μέσα στην τάξη.
